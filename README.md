# ToDo List (React)
Простое одностраничное приложение To Do List, реализованное на **React** без сторонних UI-библиотек, с использованием **ванильного CSS**.

## Функциональность
- Просмотр списка задач
- Добавление новой задачи
- Удаление задачи
- Редактирование задачи (через модальное окно)
- Поиск / фильтрация задач
- SPA-архитектура
- Разделение на отдельные компоненты

## Технологии
- React (Create React App)
- JavaScript (ES6+)
- CSS (vanilla)
- Git

## Запуск проекта
1. Установить зависимости
`npm install`

2. Запустить проект
`npm start`

Приложение будет доступно по адресу http://localhost:3000
